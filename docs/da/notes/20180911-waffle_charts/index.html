Vaffel diagram

eller; kvadratisk lagkagediagram

R
Visualisering
Forfatter

Christian Knudsen

Udgivet

11. september 2018

En forholdsvis populær type diagram. Det er ikke rigtigt en favorit hos mig, men jeg kan godt se hvordan det gør det lettere at forstå for folk der ikke er så rutinerede i at læse diagrammer eller tabeller.

Og så er min lette afsmag nok påvirket af at det er blevet brugt virkelig meget i ligegyldige “infografikker”.

Det er et gitter hvor hver celle repræsenterer et tal eller lignende. Og gøres ofte med ikoner i cellerne.

Lad os lave et eksempel:

library(ggplot2)
library(waffle)

vec <- c(`Category 1 (10)`= 10 , `Category 2 (20)`= 20,
              `Category 3 (25)`= 24, `Category 4 (16)` = 16)

waffle(vec/2, rows=3, size=0.1, 
       colors=c("#c7d4b6", "#a3aabd", "#a0d0de", "#97b5cf"), 
       title="Fire forskellige kategorier af noget", 
       xlab="1 celle = 2 noget")
Warning: Using `size` aesthetic for lines was deprecated in ggplot2 3.4.0.
ℹ Please use `linewidth` instead.
ℹ The deprecated feature was likely used in the waffle package.
  Please report the issue to the authors.

En lille irritation er at waffle insisterer på at stave “colours” forkert.

Funktionen tager en navngiven vektor af værdier, rows angiver antallet af rækker af celler eller blokke. Default er 10.

En af standardvisningerne er et 10x10 gitter, hvor hver celle repræsenterer 1% af totalen:

waffle(vec/sum(vec)*100)

Yderligere et irritationsmoment. waffle afrunder vektorens værdier. Og derfor er der kun 98 celler. Så du er nødt til at manipulere din vektor, så du sikrer dig at du får 100 celler.

Hvad hvis du vil have indsat andet end farvede kvadrater?

Funktionen tager et par argumenter der gør det muligt:

use_glyph glyph_size

First, we’ll need the library extrafont

library(extrafont)

We’ll also need to have the “awesomefonts” installed. It can be downloaded from:

http://maxcdn.bootstrapcdn.com/font-awesome/4.3.0/fonts/fontawesome-webfont.ttf

This should be easier if you are on a desktop machine. As I’m running this through my own installation of RStudio on a remote server, it was a bit more difficult.

I needed to place the “fontawesome.ttf” file in the “/usr/share/fonts/truetype/fontawesome” directory.

Then, running R as superuser on the commandline, I imported the extrafont library, and then ran “font_import()”.

But then it worked!

There is now a long list of 593 different icons, that can be used. If you want a list, just run fa_list().

And now, we can make a waffle chart with the glyph of our choice.

waffle(vec/2, rows=4, use_glyph = “wifi”)

Warning: Removed 1 rows containing missing values (geom_text).

We can change the colours:

library(RColorBrewer) waffle(vec/2, rows=4, use_glyph = “wifi”, colors=brewer.pal(4,“Set1”))

Adjust the size

waffle(vec/2, rows=4, use_glyph = “wifi”, colors=brewer.pal(6,“Set1”), glyph_size=5)

But that does not look very good.

waffle is based on ggplot, so we have access to the full range of functionality. But not all of them are going to look good in this context:

waffle(vec/2, rows=4, use_glyph = “wifi”, colors=brewer.pal(4,“Set1”)) + geom_label(label=“42”, size = 3)

Warning: Removed 1 rows containing missing values (geom_text).

If we install a font that supports it, we even get access to the large number of UTF-8 glyphs. Here is a favorite of mine:

waffle(vec/2, rows=4, colors=brewer.pal(4,“Set1”)) + geom_label(label=sprintf(“F427”), size = 8)

Which of course requires you to have a font on your computer that supports penguins.

Here is one: http://users.teilar.gr/~g1951d/Symbola.zip
